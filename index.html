Projects

Connect Four Super
May 2024

A complete re-write of a simple single-player Connect 4 game originally written in vanilla JS/HTML/CSS.

The re-write took place over the course of four individual projects (phases) culminating in this one.

  • Phase 1 - Rewrite original Connect 4 game in vanilla JS/HTML/CSS but leveraging latest learnings.
  • Phase 2 - Refactor original flat code into OOP structure.
  • Phase 3 - Rewrite in React (maintaining OOP structure of game model.
  • Phase 4 - Replicate and refactor client-side game logic to run server-side in a node.js/express.js app.
  • Phase 5 (Final) - Refactor React front-end to leverage new server-side game logic.

Features:

  • Local Multiplayer
  • AI (rudimentary) Players
  • Server-Authoritative Game State + Polling Game Updates

Stack:

  • React Front-End
  • Typescript / Node.js / Express.js Backend
  • PostgreSQL DB

IMPORTANT: All elements of the app are hosted on free resources.

  • The PostgreSQL DB is hosted on Render.com and expires on: August 15, 2024.
  • The Express.js backend is hosted on Render.com and requires 2-3 minutes to warm up.
  • The React front end is hosted on Surge.sh with no guarantee of availability (but should be infinite?).
